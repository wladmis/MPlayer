Příloha A. Mailing lists

There are some public mailing lists on MPlayer. Unless explicitly stated otherwise the language of these lists is English. Please do not send messages in other languages or HTML mail! If you send an overly big attachment and your mail gets rejected, put the attachment up for download somewhere instead. Click the links to subscribe. On the mailing lists, the Netiquette rules about writing and quoting apply just as on usenet. Please follow them, it makes the life of those who read your mails a lot easier. If you do not know them please read HOWTO Edit Messages.

  • MPlayer-announce

    List for MPlayer announcements. Subscribe here if you want to get security and release announcements.

  • MPlayer-users

    General support mailing list for questions related to MPlayer. Please read the whole documentation before posting your problems and feature requests here. MPlayer bug reports belong on this list as well, but read the known bugs and bug reporting section first.

  • MPlayer-felhasznalok (Hungarian)

    Hungarian language list. Topic? We'll see about it... mostly flame and RTFM questions up to now :(

  • MPlayer-Matrox

    List for questions related to Matrox graphics cards, like mga_vid, Matrox's official beta drivers (for XFree86 4.x.x), matroxfb-TVout stuff.

  • MPlayer-dvb

    Things related to the hardware decoder card called DVB (not DXR3!).

  • MEncoder-users

    General support mailing list for questions related to MEncoder. Please read the whole documentation before posting your problems and feature requests here. MEncoder bug reports belong on this list as well, but read the known bugs and bug reporting section first.

  • MPlayer-dev-eng

    This list is about MPlayer development! Talking about interface/API changes, new libraries, code optimization, configure changes is ontopic here. Send patches but not bug reports, user questions, feature requests or flames here to keep the list traffic low.

  • MPlayer-cvslog

    All changes in MPlayer code are automatically sent to this list. Only questions about these changes belong here (if you do not understand why a change is required or you have a better fix or you have noticed a possible bug/problem in the commit).

  • MPlayer-DOCS

    This list is about MPlayer documentation and homepage discussions. Documentation and homepage related CVS logs are also sent here.

  • MPlayer-translations

    This list is about MPlayer documentation and homepage translation discussions. Documentation and homepage translation CVS logs are also sent here.

  • MPlayer-cygwin

    List for discussion about MPlayer's Cygwin and MinGW port.

  • MPlayer-G2-dev

    List for discussion about MPlayer G2, the next generation video player.

  • MPlayer-mirror

    List for discussion about MPlayer mirror server infrastructure.

  • MPlayer-MWN (INACTIVE)

    List for discussion about the MPlayer Weekly News releases.

  • MPlayer-os2 (INACTIVE)

    List for discussion about MPlayer's OS/2 port.

  • MPlayer-qt-codecs-porting (INACTIVE)

    List for discussion about QuickTime codec support.

Poznámka

You can reach the searchable mailing list archives at http://www.mplayerhq.hu/cgi-bin/s-arch.cgi.
