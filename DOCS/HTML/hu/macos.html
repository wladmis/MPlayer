6.8. Mac OS

Csak a Mac OS X 10.2-es verzió és utódai támogatottak a hivatalos MPlayer forrásban. Régebbi verziók támogatásához javításokat szívesen fogadunk!

Az Apple saját GCC 3.x-es fordítója ajánlott, legfőképp, ha libavcodec-et is használunk, mert azt az Apple módosított GCC 2.95.x verziója nem támogatja a C99-es szintaxist.

Mindezek mellett még egy Aqua GUI is elérhető az MPlayerhez, melyet egy előre lefordított lejátszó binárissal együtt tölthetünk le az MPlayerOSX csapat oldaláról.
